CURRÍCULUM VIRTUAL - ARNALDO ESPINOLA RAMÍREZ

Estructura del proyecto:
- index.html
- css/styles.css
- js/main.js
- assets/profile.jpg
- assets/cv-referencia.jpg

Cómo usar:
1. Descomprimí la carpeta ZIP.
2. Abrí el archivo index.html en tu navegador.
3. Para subirlo a internet, podés cargar toda la carpeta a cualquier hosting.

Funciones incluidas:
- Diseño moderno en tonos lila oscuro y azul oscuro
- Animación de carga inicial
- Fondo animado con partículas
- Texto dinámico con efecto typewriter
- Animaciones al hacer scroll
- Skills con barras animadas
- Botón para imprimir/guardar como PDF
- Diseño responsive para celular y PC

Personalización rápida:
- Cambiá textos y datos en js/main.js
- Cambiá colores en css/styles.css dentro de :root
- Reemplazá la foto en assets/profile.jpg
